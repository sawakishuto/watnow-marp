## Marp とは？

マークダウン形式でプレゼンテーションを作成できるツールです。

- **シンプルな記法** - 普段のマークダウンがそのまま使える
- **バージョン管理** - Git での差分管理が容易
- **複数形式に出力** - HTML / PDF / PPTX に対応

> 「コードを書くように、スライドを書く」

---

## このテーマの特徴

### Clean Blue Line デザイン

右側の **青いライン** がアクセントになり、スライド全体に統一感を与えます。

### 読みやすさ重視

- 適切なフォントサイズと行間
- 控えめな装飾でコンテンツに集中
- 階層が明確な見出しスタイル

---

## テキストの装飾

通常のマークダウン記法で装飾できます。

- **太字** は重要なキーワードに使用
- _斜体_ は補足的な情報に
- ~~取り消し線~~ は修正履歴の表現に

### 強調表示

**重要なポイント** はテーマカラーの青色で表示されます。

---

## コードの表示

```javascript
// Marp でスライドを作成
const createSlide = (content) => {
  return {
    title: content.heading,
    body: content.text,
    theme: "clean-blue-line",
  };
};
```

シンタックスハイライトで可読性を確保しています。
